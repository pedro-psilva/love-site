Coloque os arquivos .mp3 NESTA pasta (musicas/) com EXATAMENTE
estes nomes (tudo minúsculo, com hífen, sem acento):

  1. baby-im-yours.mp3       ->  Baby I'm Yours   (toca primeiro ao entrar)
  2. r-u-mine.mp3            ->  R U Mine?
  3. i-wanna-be-yours.mp3    ->  I Wanna Be Yours

Assim o player toca a MÚSICA COMPLETA (não é mais preview do Spotify).

Depois de adicionar os arquivos, publique de novo rodando, na pasta do site:

  git add -A
  git commit -m "Adiciona musicas"
  git push
  netlify deploy --prod --dir .
